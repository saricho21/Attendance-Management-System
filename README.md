# Attendance Management System
## Project Overview
The Attendance Management System is a C++ application developed to help schools and universities manage student attendance records efficiently. The system allows users to record attendance, search student records, generate absence reports, and calculate attendance percentages.

## Objectives
* Record student attendance
* Generate attendance reports
* Track absent students
* Calculate attendance percentages
* Improve record management efficiency

## Features
### 1. Mark Attendance
Users can enter:
* Student ID
* Student Name
* Attendance Status (Present/Absent)

### 2. Display Attendance Records
Displays all recorded attendance information.

### 3. Search Student
Searches attendance records using Student ID.

### 4. Absence Report
Displays all students marked absent.

### 5. Attendance Percentage
Calculates a student's attendance percentage based on recorded attendance.

### 6. Exit System
Allows users to safely close the application.
